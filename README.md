# FORO-HUB
Este es nuestro desafío, llamado ForoHub: en él,
vamos a replicar este proceso a nivel de back end 
y, para eso, crearemos una API REST usando Spring.

## 1. Spring Boot
2. Spring Boot Security
3. PostgreSQL,
4. JPA
5. Hibernate
6. Rest API



## Herramientas

1. Java
2. Maven
3. PostgreSql - 16.x.x


El app define los siguientes APIs CRUD.

    POST /curso
    
    POST /topico

    GET /topico

    GET /topico/{id}

    UPDATE /topico/{id}
    
    DELETE /topico/{id}
    
    PUT /topico/{id}/respuestas
    
    GET /respuesta
    
    PUT /respuesta/{id}
